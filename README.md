# 环境数据信息平台（Qt）

本项目实现了“环境数据信息平台”核心功能模块，包含登录与用户管理、实时数据采集与展示、历史查询统计、异常报警、数据导出、系统设置等。

## 运行说明
- 默认管理员账号：`admin` / `admin123`
- 数据库文件自动保存在系统应用数据目录（Qt `AppDataLocation`）

## 功能模块
- 登录/注册/找回密码（MD5存储）
- 用户管理（管理员可新增/删除/切换角色/重置密码）
- 实时数据采集（模拟传感器 + 定时器）
- 历史查询 + 统计（最小/最大/平均值）
- 异常报警（可配置阈值、报警间隔、提示音）
- 数据导出（历史/统计/报警 -> CSV）
- 系统设置（刷新频率、跟随系统主题 qt6ct、数据库备份/恢复、历史数据清理）

## 构建
使用 Qt Creator 打开 `CMakeLists.txt` 即可编译运行（Qt 5.15/Qt 6 均可）。
